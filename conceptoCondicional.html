    ¿Qué es un condicional?

    Los condicionales son métodos que se emplean en códigos de Java para controlar el flujo de ejecución del programa. Esto te permite ejecutar diferentes bloques de código según si una condición lógica se cumple o no. A su vez consigue que el programa se adapte y tome diferentes decisiones, siendo sus estructuras más comunes "If", "Else If" y "Else". Un condicional, permite establecer una serie de reglas al interior de nuestro programa, que nos ayudan a determinar qué acciones llevar a cabo dadas ciertas circunstancias. Por ejemplo, si queremos decidir cuándo dar acceso a un usuario, dependiendo de si el nombre de usuario y contraseña son correctos.


    Usos

    1. Ejecutar código de forma selectiva
      • Permiten que una parte del código se ejecute solo cuando se cumpla una condición específica. Por ejemplo, mostrar un mensaje de bienvenida solo si el usuario está registrado.
    2. Tomar decisiones
      • Son la base de la lógica de decisiones en un programa, permitiendo que el programa siga diferentes caminos (ejecutando distintos bloques de código). Por ejemplo, si el saldo es suficiente, se completa la compra; si no, se rechaza.
    3. Crear software interactivo
      • Permiten que las aplicaciones reaccionen a diferentes escenarios o a las acciones del usuario, mejorando la funcionalidad y la experiencia del usuario.

    IF

    La sentencia If es la primera sentencia de condición que se le viene a la cabeza a cualquiera, ya que es la más utilizada en codigos de programación. Este sentencia se encarga de ser el inicio de todas las sentencias execpto de la sentencia switch, no tendría sentido empezar con un Else-If si no exixte un If primero. La sentencia If es la encargada de separar las instancias según las condiciones que se hayan instanciado dentro de la sentencia.

    ELSE-IF

    La sentencia If-Else se instancia despues de la sentencia If y su función es volver a separar las instancias que no cumplían las condiciones del If, pero si que cumplen las del Else-If. De aqui la dependencia de esta sentencia, sin If, no existen else if. Se pueden acercadenas de else-if para dar crear varios flujos de opciones y actuan en orden, es decir, que se efectuan los else-if por orden, aunque se cumplan else-if despues del ya ejecutado, no se ejecutaran más.

    IF-ELSE IF

    Esta sentencia es un añadido a un aprevia sentencia If. La sentencia If-Else If se encarga de añadir una condición más al if al que sigue. Es una sentencia poco utilizada debido a que en la la misma sentencia If se pueden agregar distintas condiciones.

    ELSE

    La sentencia final en un condicional de tipo If es un Else. Este se encarga de seleccionar a todas las instancias que no han cumplido ninguna de las anteriores sentencias.

    SWITCH

    No encontramos ante la otra posible forma de generar un condicional.El switch es una estructura de control de flujo que permite ejecutar diferentes bloques de código según el valor de una variable. Se evalúa una expresión (generalmente de tipo int, String, char, o enum) y se compara con varios casos (case). La ejecución se dirige al bloque de código que corresponda al valor encontrado.

    Partes de un Switch

    • Comienzo de la sentencia
      • Switch(variable)
    • Case
      • Son los distintos casos que pueden ocurrir en el condicional
      • Cada case termina en switch si solo se quiere que se ejecute el case escogido
      • Normalmente se utiliza un Scanner para que el usuario escoja la opción a ejecutar
