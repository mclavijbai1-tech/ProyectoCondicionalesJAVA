    • Tipo IF

    Aqui tenemos un ejemplo de la condicional IF 

                if (Edad >= 18 ){

            System.out.println("Mayor de edad")
           
            }

    Aqui se puede ver que si edad es mayor o igual se le considera mayor de edad

    Tipo IF Else

    Aqui tenemos un ejemplo de la condicional IF ELSE

                if (Edad >= 18 ){

            System.out.println("Mayor de edad")
           
            } else {

                System.out.println("Menor de edad")

            }

    Aqui se puede ver que si no es mayor o igual a 18 en vez de quedarse callado te pondra menor de edad

    Tipo Else IF

    Aqui tenemos un ejemplo de la condicional ELSE IF

                if (Edad >= 18 ){

            System.out.println("Mayor de edad")
           
            } else if (Edad  >= 16) {

                System.out.println("Joven responsable")

            } else {
                System.out.println("Menor de edad")
            }

    Aqui se puede ver que el else if lo que hace es crear mas filtros para poder saber mejor la edad del usuario

    Tipo Switch

    Aqui tenemos un ejemplo de la condicional Switch

                 int DiaSemana = 5;

            switch (DiaSemana){

                case 1:
                    System.out.println("Lunes");
                    break;

                case 2:
                    System.out.println("Martes");
                    break;

                case 3:
                    System.out.println("Miercoles");
                    break;

                case 4:
                    System.out.println("Jueves");
                    break;

                case 5:
                    System.out.println("Viernes");
                    break;

                case 6:
                    System.out.println("Sabado");
                    break;

                case 7:
                    System.out.println("Domingo");
                    break;
            }

    Aqui podemos ver que dependiendo de la variable lo que hace el switch es ir a una rama o a otra
